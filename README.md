pantalla1
Crea una pantalla de inicio para una app de streaming en Flutter. El fondo debe ser negro absoluto. En el centro, coloca una letra 'N' mayúscula de color rojo (#E50914), utilizando un Transform.scale para que sea 1.8 veces más ancha de lo normal, dándole un aspecto robusto y cinematográfico. Debajo de la letra, añade un botón rectangular con bordes ligeramente redondeados (radio de 4) de color rojo con el texto 'INICIAR' en negrita y letras blancas. Al presionar el botón, debe navegar a la pantalla de login.

pantalla2
Diseña una pantalla de 'Inicio de Sesión' en Flutter con estética oscura. El título debe estar en la parte superior izquierda en blanco y negrita. Crea un widget reutilizable para los campos de entrada que consista en un contenedor con fondo rojo (#E50914), esquinas rectas y un icono blanco a la izquierda; el texto dentro debe ser blanco. Necesito dos de estos campos: uno para 'Correo electrónico' y otro para 'Contraseña'. Debajo, añade dos botones de ancho completo: el primero de color azul fuerte para 'INICIAR SESIÓN' y el segundo de color verde vibrante para 'REGISTRARSE' que redirija a la siguiente pantalla

pantalla3
Crea una pantalla de 'Registro' en Flutter que use un SingleChildScrollView para evitar errores de desbordamiento. La pantalla debe tener 5 campos de texto verticales con fondo rojo y letras blancas (Usuario, Celular, Correo, Contraseña y Confirmar Contraseña). Cada campo debe incluir un icono descriptivo blanco. Al final del formulario, coloca un botón grande de color rojo que diga 'CREAR CUENTA' y, debajo de este, un TextButton sencillo en color blanco que diga 'Cancelar' para regresar a la pantalla anterior. Incluye la flecha de navegación hacia atrás en la AppBar negra.
